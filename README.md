# Bistro FDI - Práctica 2
Proyecto de Aplicaciones Web 2025/2026.
Grupo 6.

## Instrucciones de instalación local:
1. Clonar este repositorio en `htdocs`.
2. Duplicar `includes/db.example.php` y renombrarlo a `includes/db.php`.
3. Configurar en `db.php` las credenciales locales de MySQL.
4. Importar el archivo `bistro.sql` en phpMyAdmin.
